# Asset Folders

Use this directory for dashboard images.

- `public/assets/logos/`: app logos (example: `medalverse-logo.svg`)
- `public/assets/banners/`: header/hero backgrounds (example: `dashboard.svg`)
- `public/assets/cards/`: event card images (example: `card-01.svg` ... `card-05.svg`)

You can replace the example files with real PNG/JPG/SVG files while keeping the same names.
